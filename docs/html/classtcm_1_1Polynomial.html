template<class T>
tcm::Polynomial< T >::Polynomial ( Container const &  coef)
inlinenoexcept

Constructs a Polynomial from given coefficients.

In other words, this function assumes the dense representation of coefficients. The constructed polynomial will look like this [ p(x) = {i = 0}^{{{coef.size()}}} {{coef[i]}} x^i. ] All trailing zero coefficients are removed with the remove_redundant() function. If coef.size() is 0, then the underlying _coef vector will contain a single zero.

template<class T>
auto tcm::Polynomial< T >::remove_redundant ( ) -> void
inlinenoexcept

Removes trailing zero coefficients from the Polynomial.

Example:

1)
[1, 2, 1, 0] ---> [1, 2, 1]
i.e. 1 + 2x + x^2 + 0x^3 ---> 1 + 2x + x^2.
2)
[0, 0, 0] ---> [0]
i.e. 0 + 0x + 0x^2 ---> 0.
