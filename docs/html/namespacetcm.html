Adds a term \( c / (x - b) \) to an expr \( \sum_{i=0}^N a_i x^i / \prod_{i=0}^N(x - b_i) \).

Say we have \( \text{num} / \text{den} = \sum_{i=0}^N a_i x^i / \prod_{i=0}^N(x - b_i)\). Then

\[ \frac{\text{num}}{\text{den}} + \frac{c}{x - b} = \frac{\sum_{i=0}^N a_i x^i}{\prod_{i=0}^N(x - b_i)} + \frac{c}{x - b} = \frac{(x - b)\cdot\sum_{i=0}^N a_i x^i + c\cdot\prod_{i=0}^N(x - b_i)} {(x - b)\prod_{i=0}^N(x - b_i)}. \]

This means:

  • \( \text{num} \gets \text{num} \cdot (x - b),\)
  • \( \text{num} \gets \text{num} + c \cdot \text{den},\)
  • \( \text{den} \gets \text{den} \cdot (x - b)\).

noexcept

Computes \( f(E) \).

Calculates the Fermi-Dirac distribution:

\[ f(E) := \frac{1}{\exp(\frac{E - \mu}{k_{\text{B}}T}) + 1}. \]

All parameters are templated to allow the use of floating point numbers with different precision. The result, however, is (implicitely) converted to _F, i.e. the type of E.

Template Parameters
_FRepresents a real/compex field.
_RRepresents a real field.
Parameters
mu\( \mu \) is the chemical potential
t\( T \) is the temperature
kb\( k_\text{B} \) is the Boltzmann constant.
Returns
\( f(E) \). Should not throw.
