Detailed Description

Defines the Ring interface. From mathematics, a ring is a set with binary operations + and * satisfying the following axioms:

  • is abelian under additions, i.e. is associative ((a + b) + c = a + (b + c)), commutative (a + b = b + a), has an additive identity (element 0), has additive inverse (-a for every element a).
  • is a monoid under multiplications, i.e. is associative ((a * b) * c = a * (b * c)), has the multiplicative identity (element 1).
  • multiplication is distributive with respect to addition, i.e. a * (b + c) = (a * b) + (a * c) and (b + c) * a = (b * a) + (c * a).
