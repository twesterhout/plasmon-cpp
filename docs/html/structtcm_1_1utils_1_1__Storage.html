Detailed Description

template<class _Tp, class _Alloc>
struct tcm::utils::_Storage< _Tp, _Alloc >

Contiguous chunk of memory.

This class is like a std::vector except that it allows no resizing and is thus smaller in size by sizeof(pointer).
